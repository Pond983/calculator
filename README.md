# calculator
This is the Calculator program.
This calculate input formula.

# How to use.
To use it, compile all c files in source_codes directory by gcc compiler.
You also use calculator.sh to turn on calculator easily.

```shell

./calculator.sh
>

```

# Specifications
You can use four opelands and curl bracket in formula.
- "+"
- "-:
- "*"
- "/"
- "(" ")"

if you want to stop program, type "quit" and end the program.

```shell

> quit

```
